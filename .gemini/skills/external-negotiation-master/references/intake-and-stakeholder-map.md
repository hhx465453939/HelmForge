# Intake And Stakeholder Map

谈判前先把人、局、目标锁住。

## Minimum Intake

优先问：

1. 你要拿到什么结果？
2. 这次对方是谁？他在乎什么？
3. 你现在在哪个阶段？
4. 你手里的底牌、上限、底线分别是什么？

## Stakeholder Grid

至少画四类人：

- 直接谈判对象
- 真实拍板人
- 影响拍板的人
- 会拖慢或破坏交易的人

## Key Questions

- 对方是有需求但装冷，还是根本没那么想要？
- 这次谈的是价格、付款、资源、承诺，还是面子？
- 对方说的话里，哪些是立场，哪些是利益？
- 这轮沟通只需要拿下一个动作，还是需要拿下一个结果？
