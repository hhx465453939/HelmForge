# 财务比率体系

四大类财务比率：偿债 / 盈利 / 营运 / 成长。每个比率都要"公式 + 健康区间 + 行业差异 + 常见误读"四件套一起看，脱离行业和口径谈比率都是耍流氓。

## 1. 偿债能力比率

衡量公司偿还债务的能力，关注短期与长期两个层次。

### 1.1 流动比率 Current Ratio

    流动比率 = 流动资产 / 流动负债

- **一般健康区间**：1.5 - 2.5
- **过低（< 1.0）**：短期偿债承压
- **过高（> 3.0）**：可能存在存货 / 应收堆积、资金使用效率低
- **行业差异**：
  - 零售 / 餐饮：0.8 - 1.5（周转快、现金好）
  - 制造：1.5 - 2.5
  - 房地产：0.8 - 1.5（存货即在建楼盘，含义特殊）
  - 银行：无意义（结构完全不同）

### 1.2 速动比率 Quick Ratio

    速动比率 = (流动资产 - 存货 - 预付款项) / 流动负债

- **健康区间**：0.8 - 1.2
- 剔除变现慢的存货和预付，比流动比率更严格

### 1.3 现金比率 Cash Ratio

    现金比率 = (货币资金 + 交易性金融资产) / 流动负债

- **健康区间**：0.2 - 0.5
- **警戒**：< 0.1 时短期流动性极其紧张
- **异常**：> 1.0 且同时有大量有息负债 → 存贷双高，警惕舞弊或资金被占用

### 1.4 资产负债率 Debt-to-Assets

    资产负债率 = 总负债 / 总资产

- **一般健康区间**：40% - 60%
- **行业差异极大**：
  - 银行 / 保险：85% - 95%（负债经营，正常）
  - 房地产：60% - 85%
  - 制造：40% - 60%
  - 消费品 / 医药：30% - 50%
  - SaaS / 轻资产互联网：< 40%

### 1.5 利息保障倍数 Interest Coverage

    利息保障倍数 = EBIT / 利息费用
    （EBIT = 利润总额 + 利息费用）

- **健康**：> 3
- **警戒**：< 1.5（利润几乎全被利息吃掉）
- **危险**：< 1（利润不够付利息）

### 1.6 现金短债比

    现金短债比 = 货币资金 / 短期有息负债

- **健康**：> 1.0
- 房地产行业"三道红线"之一（要求 > 1.0）

## 2. 盈利能力比率

衡量赚钱能力。要区分口径：毛利 → 营业利润 → 净利润 → 归母净利润。

### 2.1 毛利率 Gross Margin

    毛利率 = (营业收入 - 营业成本) / 营业收入

- **反映**：产品定价权 + 成本控制能力
- **行业基准差异极大**：
  - 白酒 / 品牌奢侈品：70% - 90%
  - 创新药：60% - 90%
  - SaaS：60% - 85%
  - 互联网平台：40% - 70%
  - 品牌消费品：30% - 60%
  - 一般制造：15% - 30%
  - 大宗商品 / 建材 / 钢铁：5% - 20%
  - 零售 / 分销：5% - 20%
- **关注**：单期跳升要问原因（会计口径变更？产品结构？）

### 2.2 营业利润率 Operating Margin

    营业利润率 = 营业利润 / 营业收入

- **反映**：主营业务盈利能力（剔除营业外损益）
- **健康**：制造业 5-10%、消费品 10-20%、软件 / 品牌 20-40%

### 2.3 净利率 Net Profit Margin

    净利率 = 净利润 / 营业收入

- **反映**：整体盈利能力（含税负、非经常损益）
- **注意**：净利率高可能是非经常性收益（补贴 / 资产处置 / 投资收益）贡献，要剥离

### 2.4 ROA 总资产收益率

    ROA = 净利润 / 平均总资产
    平均总资产 = (期初总资产 + 期末总资产) / 2

- **反映**：单位资产创造利润的能力
- **健康区间**：制造 3-8%、消费品 8-15%、软件 / 平台 10-25%
- **杜邦分解**：ROA = 净利率 × 总资产周转率

### 2.5 ROE 净资产收益率

    ROE = 归母净利润 / 平均归母净资产
    平均归母净资产 = (期初归母净资产 + 期末归母净资产) / 2

- **反映**：股东资金的赚钱效率，是最核心的盈利指标
- **健康区间**：
  - 巴菲特参考线：> 15% 长期可持续
  - 中国 A 股平均：8% - 12%
  - 优秀公司：15% - 25%
  - **> 30% 长期**：要仔细追问是否杠杆过高或异常
- **杜邦分解**：ROE = 净利率 × 总资产周转率 × 权益乘数（详见 `dupont.md`）

## 3. 营运能力比率

衡量资产使用效率、经营周期效率。

### 3.1 总资产周转率 Asset Turnover

    总资产周转率 = 营业收入 / 平均总资产

- **健康**：制造 0.6 - 1.2、零售 1.5 - 3.0、SaaS 0.4 - 0.8
- 反映每 1 元资产撬动的营收

### 3.2 应收账款周转率与周转天数

    应收账款周转率 = 营业收入 / 平均应收账款
    应收账款周转天数 = 365 / 周转率

- **健康区间**：视账期而定，一般制造业 60-90 天
- **警戒**：周转天数持续拉长 → 客户回款能力恶化 / 收入虚增
- **拉长的常见原因**：
  - 给大客户让步账期换订单
  - 收入是"塞渠道"塞出来的
  - 关联方大客户拖欠

### 3.3 存货周转率与周转天数

    存货周转率 = 营业成本 / 平均存货
    存货周转天数 = 365 / 周转率

- **健康区间**：快消 30-60 天、耐用品 60-120 天、房地产不适用（用去化率）
- **警戒**：周转天数拉长 + 跌价准备不计提 → 存货减值风险

### 3.4 应付账款周转率与周转天数

    应付账款周转率 = 营业成本 / 平均应付账款
    应付账款周转天数 = 365 / 周转率

- 越长表示对供应商话语权越强（在健康范围内），但过长可能是流动性紧张

### 3.5 现金循环周期 Cash Conversion Cycle（CCC）

    CCC = 存货周转天数 + 应收账款周转天数 - 应付账款周转天数

- **含义**：从付出现金进货，到最终收回现金，全程需要多少天
- **越短越好**：CCC 为负（如某些平台电商 / 大零售）意味着占用供应商资金经营
- **警戒**：CCC 大幅拉长 → 现金流即将紧张

## 4. 成长能力比率

衡量增长速度与质量。

### 4.1 营业收入增长率

    营收增长率 = (本期营收 - 上期营收) / 上期营收

- 需要与行业整体增速对比看相对表现
- 一次性大项目 / 并表引起的跳增要单独列示

### 4.2 净利润增长率

    净利增长率 = (本期净利润 - 上期净利润) / |上期净利润|

- 若上期为负 / 微利，本期为正，比率意义有限，用绝对值增量描述更清楚

### 4.3 归母净利润增长率

同上，用归属母公司股东净利润口径，剔除少数股东影响

### 4.4 可持续增长率（Sustainable Growth Rate）

    可持续增长率 = ROE × (1 - 分红率)

- 反映在不增发股本、不改变财务结构条件下能维持的最高增长率
- 实际增长 > 可持续增长 → 依赖外部融资
- 实际增长 < 可持续增长 → 现金积累，可能需要更积极的资本运作

## 5. 计算示例

假设 A 公司 2023 年：
- 营收：120 亿
- 净利润：12 亿（归母 11 亿）
- 平均总资产：150 亿
- 平均归母净资产：60 亿
- 流动资产：50 亿，其中存货 10 亿、应收 15 亿
- 流动负债：30 亿
- 货币资金：8 亿
- 短期借款：5 亿

推算：
- 净利率 = 12/120 = 10%
- 总资产周转率 = 120/150 = 0.80
- ROA = 12/150 = 8%
- ROE（归母口径）= 11/60 = 18.3%
- 权益乘数 = 150/60 = 2.5
- 杜邦验算：10% × 0.80 × 2.5 ≈ 20%（与 ROE 差异由归母口径与少数股东权益导致，需口径对齐）
- 流动比率 = 50/30 = 1.67 ✅
- 现金比率 = 8/30 = 0.27 ✅
- 现金短债比 = 8/5 = 1.6 ✅

## 6. 常见误读与注意事项

1. **平均值 vs 期末值**：分子是流量（利润 / 营收），分母若是存量（资产 / 权益），一律用期初期末平均值。跨期比较时口径要一致。
2. **归母口径 vs 全部口径**：ROE 分子分母要么全归母，要么全全部，不能混。
3. **少数股东权益**：合并报表时，如果少数股东占比高，不用归母口径 ROE 会失真。
4. **一次性损益**：净利润里如果含大额政府补贴 / 资产处置收益，要单独扣除后再看趋势。
5. **会计政策变更**：追溯调整期间的比率跳变要小心，看财报附注是否披露。
6. **行业属性**：银行、保险、房地产、券商等特殊行业，通用比率大部分不适用，见 `industry-benchmarks.md`。
7. **单期 vs 趋势**：一期数据不足以判断，至少 3 期。
8. **比率间的联动**：不要孤立看单个比率。例如流动比率高但现金比率低，说明流动资产里现金少存货应收多，风险其实不低。

## 7. 快速健康评估打分模板

给每个维度一个红黄绿灯，快速形成初步判断：

| 维度 | 主指标 | 绿灯 | 黄灯 | 红灯 |
|---|---|---|---|---|
| 短期偿债 | 现金比率 | > 0.3 | 0.1 - 0.3 | < 0.1 |
| 长期偿债 | 资产负债率 | 行业中位 ±10pct | ±20pct | > +20pct |
| 盈利 | ROE（3年均值）| > 15% | 8-15% | < 8% |
| 盈利 | 毛利率趋势 | 稳定或改善 | 微降 | 连续两期显著下滑 |
| 营运 | 现金循环周期 | 缩短 / 稳定 | 略拉长 | 大幅拉长 |
| 现金流质量 | CFO / 净利润 | > 0.9 | 0.5 - 0.9 | < 0.5（连续两期）|
| 成长 | 营收增速 | > 行业 | ≈ 行业 | < 行业且连续下滑 |

红灯 ≥ 2 个 → 财务健康显著承压，进一步展开风险点分析。
