# /strategy-cfo

调用 `strategy-cfo` skill，进入商业战略 × 财商融合的中长期决策模式。

## 何时使用

当你需要处理"要不要做、做的话怎么做、值多少钱、钱从哪来、什么时候退出"这类中长期（3-10 年）战略决策问题时：

- **业务组合决策**：多业务线要不要砍、要不要新建、主业要不要转型
- **重大资本开支（Capex）**：新工厂 / 大 IT 系统 / 扩产能，做 NPV / IRR / 敏感性
- **融资决策**：下一轮融多少、按什么估值、股权 vs 债权、轮次节奏
- **并购决策（M&A）**：估值区间、协同测算、支付方式、整合 IRR
- **出海 / 国际化**：地区选择、进入方式（自建 / 合资 / 收购）、资本部署顺序
- **IPO / 上市地选择**：A股 / 港股 / 美股的估值差异与后续融资便利性权衡
- **商业模式重设计**：SaaS 化、订阅制、平台化、生态化的财务影响
- **战略退出**：卖公司 / 卖业务 / MBO / 分红 / 拆分的估值权衡
- **股权激励方案的经济学**：期权池、行权价对稀释与估值的影响

## 与 executive-consultant / finance-manager / budget-architect 的边界

- **executive-consultant** 偏管理 / 组织 / 领导力 / 人事 → 回答"人和组织怎么破"
- **strategy-cfo** 偏商业战略 × 财商 → 回答"要不要做、值多少、钱从哪来、怎么退出"
- **finance-manager** 偏历史财务分析 → 回答"过去 / 现在财务上到底怎么样"
- **budget-architect** 偏未来 12 个月预算与执行 → 回答"下一年 / 下一季度钱怎么规划"

四个 skill 通常联动使用：finance-manager 给基线 → executive-consultant 给组织可行性 → strategy-cfo 决策 → budget-architect 落地。

**路由提示词**：出现"战略、商业模式、要不要做、值多少、融资、并购、M&A、投资决策、NPV、IRR、DCF、资本结构、估值、退出、业务组合、Capex"等词，默认路由到 strategy-cfo。

## 输入建议

尽量一次提供以下要素：

1. **决策类型**：业务组合 / 重大 Capex / 融资 / 并购 / 出海 / IPO / 退出 / 商业模式转型
2. **公司基本信息**：行业、规模（营收 / 员工数）、发展阶段（种子 / 成长 / 成熟 / 转型）、股权性质（家族 / VC 系 / 上市 / 央国企）
3. **战略意图**：董事会 / 大股东 / 老板到底想要什么（规模 / 利润 / 上市 / 现金退出 / 传承）
4. **历史财务基线**：至少最近 3 年营收、利润、现金流、资产负债概况
5. **市场信息**：目标市场规模、增长率、竞争格局、主要玩家的估值
6. **约束条件**：可用资本上限、股权稀释红线、董事会 / 大股东硬指标、时间窗口
7. **风险偏好**：保守 / 稳健 / 进取；能承受几年亏损

信息不全也可以先开始，`strategy-cfo` 会显式标注假设并给出置信度。

## 默认输出

- **战略选项**：2-3 个候选战略方向，各自的战略逻辑、时间窗口、关键假设
- **财务可行性**：各选项的资本需求、部署节奏、目标毛利 / 净利、关键风险
- **资本结构建议**：融资组合（股 / 债 / 可转债 / 内源）、WACC 影响、杠杆策略
- **投资回报测算**：NPV / IRR / 回收期 / 获利指数 + 敏感性 + 情景分析（乐观 / 中性 / 悲观）
- **估值影响**：当前估值 vs 实施后目标估值，稀释影响，退出通道估值区间
- **推荐路径与退出条件**：分阶段 Gate + 止损红线 + 决策提交清单
- **数据假设与依赖标注**：用户已给 / 合理假设 / 缺失数据 / 需外部专业机构复核的部分

## 相关 skill

- 组织可行性 / 变革管理 / 人才准备度 → `executive-consultant`
- 历史财务基线 / 财务比率 / 估值输入 → `finance-manager`
- 战略决策后的年度预算落地 → `budget-architect`
- 电商业务的估值特殊性（GMV/ROAS/平台锁定）→ `ecommerce-finance`
- 融资 / 并购 / 战投的实际谈判 → `external-negotiation-master`
- 交易法律文件（SPA、股东协议、境内外架构） → `global-legal-counsel`
- 市场情报 / 竞对分析 / 政策研究（作为战略输入） → `deep-research`

## 使用注意

- 所有重大战略决策必须经董事会 / 投资人复核，本 skill 只出决策 menu。
- 涉及一级 / 二级市场融资、并购、IPO 的正式估值需 IB / FA / 独立审计师加持。
- 战略溢价（卡位、防御、实物期权）如果显式出现在推荐依据中，必须量化，不能藏。
- 中国式财务现实（家族因素、市值管理、政治任务）会写进决策依据，不假装没有。
