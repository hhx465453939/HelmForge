# 行业基准对比

任何财务比率脱离行业都可能误判。这份文档给出主要行业的基准比率范围，用作快速对照。数字为经验区间（中国 A 股 + 港股大陆企业为主，兼顾美股），会随宏观周期与行业成熟度变化，建议每年更新一次。

## 使用说明

- 表格中的数字是**行业中位数附近的常见区间**，不是绝对区间
- 单个公司偏离行业不一定坏，但要能解释清楚
- 处于行业头部 / 尾部的公司偏离通常更大，可以再细分头部 / 中部 / 尾部三档看
- 特殊会计口径（银行 / 保险 / 房地产）不适用通用比率，见文末专章

---

## 1. 制造业（一般加工制造 / 装备）

| 指标 | 典型区间 |
|---|---|
| 毛利率 | 15% - 30% |
| 营业利润率 | 5% - 12% |
| 净利率 | 3% - 10% |
| ROE | 8% - 15% |
| ROA | 3% - 8% |
| 总资产周转率 | 0.6 - 1.2 |
| 流动比率 | 1.5 - 2.5 |
| 速动比率 | 1.0 - 1.5 |
| 资产负债率 | 40% - 60% |
| 应收账款周转天数 | 60 - 120 |
| 存货周转天数 | 60 - 120 |
| 现金循环周期 | 60 - 120 |
| CapEx / 营收 | 5% - 10% |

**要点**：重资产、周转慢、毛利中等。头部品牌溢价明显（如白电、工程机械头部），细分龙头毛利可到 30-40%。

**细分差异**：
- 消费电子（手机 / 白电）：毛利 15-25%，周转较快
- 装备制造（机床 / 工程机械）：毛利 25-35%，账期长
- 汽车零部件：毛利 15-25%，客户集中度高
- 化工基础材料：毛利 10-20%，周期性强

---

## 2. 零售业（超市 / 连锁 / 大卖场）

| 指标 | 典型区间 |
|---|---|
| 毛利率 | 15% - 30% |
| 营业利润率 | 2% - 6% |
| 净利率 | 1% - 4% |
| ROE | 10% - 20% |
| ROA | 3% - 8% |
| 总资产周转率 | 1.5 - 3.0 |
| 流动比率 | 0.8 - 1.5 |
| 速动比率 | 0.4 - 0.8 |
| 资产负债率 | 40% - 65% |
| 存货周转天数 | 30 - 60 |
| 应付账款周转天数 | 45 - 90 |
| 现金循环周期 | -30 - 30（可负）|

**要点**：低毛利、高周转、高杠杆是常态。占用供应商资金是核心商业模式，CCC 为负是加分项。

**细分差异**：
- 便利店 / 高频快消：CCC 常为负，毛利 20-30%
- 3C 数码零售：毛利 8-15%，周转很快
- 服装零售：毛利 40-60%，但库存风险高
- 大卖场：CCC 微负，毛利 20-25%

---

## 3. 品牌消费品（食品饮料 / 日化 / 服装品牌）

| 指标 | 典型区间 |
|---|---|
| 毛利率 | 30% - 70%（白酒 70-90%）|
| 营业利润率 | 10% - 25% |
| 净利率 | 8% - 20%（白酒 30%+）|
| ROE | 15% - 30% |
| ROA | 8% - 20% |
| 总资产周转率 | 0.6 - 1.5 |
| 资产负债率 | 20% - 40%（现金奶牛低杠杆）|
| CFO / 净利润 | > 1.0 常见 |
| 分红率 | 30% - 70% |

**要点**：品牌溢价高、现金流强、低杠杆。存货结构和渠道库存要看清（是否塞渠道）。

**细分差异**：
- 白酒：毛利 70-90%，ROE 20-30%，超强现金流
- 调味品 / 乳制品：毛利 30-45%
- 服装品牌：毛利 50-65%，库存与折扣管理是关键
- 日化：毛利 50-65%

---

## 4. 医药与创新药

| 指标 | 典型区间（成熟药企）|
|---|---|
| 毛利率 | 60% - 90% |
| 营业利润率 | 15% - 30% |
| 净利率 | 12% - 25% |
| ROE | 12% - 25% |
| 研发费用率 | 8% - 20%（创新药更高） |
| 销售费用率 | 15% - 40%（推广重资产）|
| 资产负债率 | 20% - 40% |

**要点**：毛利高但销售 / 研发费用也高。创新药未上市阶段可能长期亏损，估值不适用 DCF。

**细分**：
- 大 Pharma / 中药：净利率 15-25%
- 仿制药：毛利 50-70%，净利率 5-15%
- CXO（研发外包）：毛利 30-45%，成长性关键
- 医疗器械：毛利 40-70%，看产品分层
- Biotech 早期：净利可能大幅为负，用平台估值 + 管线估值

---

## 5. SaaS / 企业软件

| 指标 | 典型区间（成熟 SaaS）|
|---|---|
| 毛利率 | 60% - 85% |
| 营业利润率 | -10% - 25%（高增长期可为负）|
| 净利率 | -10% - 20% |
| ROE | 变化极大 |
| 营收增长率（早期）| 30% - 100%+ |
| 营收增长率（成熟）| 15% - 30% |
| 客户续费率 NDR | > 100% 优秀（120%+ 顶级）|
| CAC / LTV 比 | 1:3 起步、1:5 优秀 |
| 资产负债率 | 20% - 40% |
| Rule of 40 | 营收增速 + 营业利润率 > 40% |

**要点**：早期看增长（营收 + NDR），成熟期看盈利与自由现金流。DCF 显性期常需拉长到 8-10 年。

**关键差异**：与传统软件"卖 license"不同，SaaS 前期投入大，看 LTV 回收。

---

## 6. 房地产开发

| 指标 | 典型区间 |
|---|---|
| 毛利率 | 15% - 30%（周期波动大）|
| 营业利润率 | 5% - 15% |
| 净利率 | 3% - 10% |
| ROE | 8% - 20% |
| 资产负债率 | 60% - 85% |
| 剔除预收后的资产负债率 | 60% - 75%（红线：70%） |
| 净负债率 | 30% - 120%（红线：100%） |
| 现金短债比 | 0.6 - 2.0（红线：1.0） |
| 存货周转 | 用去化率而非天数 |

**要点**：**三道红线**（剔预资产负债率、净负债率、现金短债比）是行业硬约束。存货是在建楼盘，含义特殊。DCF 通常不适用，用 NAV。

---

## 7. 互联网平台 / 广告

| 指标 | 典型区间（成熟龙头）|
|---|---|
| 毛利率 | 40% - 70% |
| 营业利润率 | 15% - 35% |
| 净利率 | 15% - 30% |
| ROE | 15% - 30% |
| 资产负债率 | 20% - 45% |
| 现金 + 短投 / 总资产 | 30% - 60%（现金富裕） |
| CFO / 净利润 | > 1.0 常见 |

**要点**：轻资产、高毛利、高现金流。用户与 GMV 增速是关键前置指标，广告 / 电商 / 云 / 直播等分部差异大。

## 8. 银行（不适用通用比率，用银行专属指标）

| 指标 | 典型区间 |
|---|---|
| ROE | 10% - 15% |
| ROA | 0.6% - 1.2% |
| 净息差 NIM | 1.5% - 2.5%（中国大行 1.7-2.0） |
| 不良贷款率 | 0.9% - 2.0% |
| 拨备覆盖率 | 150% - 300% |
| 核心一级资本充足率 | ≥ 7.5%（监管），头部行 10%+ |
| 资本充足率 | ≥ 10.5%（监管），头部行 12-15% |
| 成本收入比 | 25% - 40% |

**要点**：负债经营是常态（资产负债率 90%+），流动比率 / 权益乘数无实义。用银行专属指标：净息差、不良率、拨备、资本充足率。

---

## 9. 保险（人身险 / 财产险）

| 指标 | 典型区间 |
|---|---|
| 综合成本率（财险）| 95% - 105%（< 100% 才承保盈利）|
| 内含价值（EV）增长率 | 10% - 20% |
| 新业务价值率 NBV Margin | 20% - 40%（寿险）|
| ROE | 10% - 18% |
| 偿付能力充足率 | ≥ 150%（红线 100%）|

**要点**：不用 PE / PB，用 P/EV（价格 / 内含价值）估值。DCF 用剩余收益模型或 FCFE。

---

## 10. 券商 / 投行

| 指标 | 典型区间 |
|---|---|
| ROE | 5% - 12%（周期性）|
| 资产负债率 | 60% - 80% |
| 净资本 / 净资产 | ≥ 40%（监管） |
| PB | 1.0 - 2.5 |

**要点**：周期性极强（牛市 vs 熊市 ROE 差 5-10 倍），估值用 PB 而非 PE，用长周期中值 ROE。

---

## 11. 港股 / 美股行业特殊性

- 美股上市中国公司：可能同时披露 GAAP 和 non-GAAP 利润，non-GAAP 剔除股权激励，差异需明确
- 港股金融地产为主，通用比率适用性同大陆
- 美股 SaaS 高估值通常隐含 5-10 年高增长假设
- 中概股 VIE 结构：股权价值归属和现金分红能力受限，估值需折价

---

## 12. 高杠杆行业（银行 / 保险 / 房地产）比率解读差异

**为什么通用比率失效**：

1. **银行 / 保险**：负债即"原材料"（存款 / 保费），高负债是商业模式必然，不是财务风险
2. **房地产**：存货 = 在建楼盘（周期 3-5 年才转化），周转天数无意义
3. **高杠杆行业风险不在杠杆本身，而在**：
   - 资产质量（不良率 / 减值 / 去化率）
   - 流动性错配（长期资产 vs 短期负债）
   - 监管约束（资本充足率 / 三道红线 / 偿付能力）

**分析建议**：
- 银行：看不良率、拨备、净息差、资本充足率
- 保险：看内含价值、新业务价值、偿付能力
- 房地产：看销售回款、去化率、三道红线、拿地强度

---

## 13. 使用注意

1. **区间是"常见"而非"合理"**：偏离并不必然坏，但要能解释
2. **数据来源**：优先用行业协会、上市公司年报中位数、券商研报
3. **更新周期**：宏观周期变化时（如利率大幅变化、行业政策转向），需要重新校准
4. **头部 / 中部 / 尾部**：龙头公司通常在各项指标上都好于行业中位数，比较时优先与"可比同规模同商业模式"公司对齐
5. **国际对比**：跨市场对比时，注意会计准则（CAS / IFRS / US GAAP）、税负、劳动力成本、估值文化差异
