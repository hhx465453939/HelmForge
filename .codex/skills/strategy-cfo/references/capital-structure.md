# 资本结构决策（Capital Structure）

资本结构 = 债务融资 vs 股权融资的比例组合。表面上是"从哪借钱"，实际上决定了 WACC、财务杠杆、破产风险、股东回报、控制权、后续融资空间——是 CFO 最核心的战略工具之一。

## 一、核心概念速览

### 1. WACC（Weighted Average Cost of Capital，加权平均资本成本）

$$
WACC = \frac{E}{V} \times R_e + \frac{D}{V} \times R_d \times (1 - T)
$$

- E = 股权市值，D = 债务市值，V = E + D
- R_e = 股权成本（用 CAPM：R_f + β × 市场风险溢价）
- R_d = 债务成本（票面利率或到期收益率）
- T = 边际所得税率（中国普通企业 25%，高新技术 15%，小微 20%）

**关键洞察**：因为债务利息可税前抵扣，所以债务的**税后成本** = R_d × (1 − T)，几乎总是低于股权成本。这是"债务便宜"的根本原因。

### 2. 财务杠杆（Financial Leverage）

杠杆率有多种定义，最常用两种：
- **Debt-to-Equity Ratio (D/E)** = 总债务 / 股东权益
- **Debt-to-Assets Ratio (D/A)** = 总债务 / 总资产

**放大效应**：
$$
ROE = ROA \times \frac{Assets}{Equity}
$$

假设 ROA = 8%，资产 100，全股权：ROE = 8%。加 50 债务（利率 5%）后：
- 资产收益 = 100 × 8% = 8
- 利息 = 50 × 5% = 2.5
- 净利润（不考虑税） = 5.5
- 权益 = 50
- ROE = 5.5 / 50 = 11%

杠杆把 ROA 8% 放大成 ROE 11%。**这是股东获得超额回报的核心机制。**

**但风险也放大**：如果 ROA 掉到 3%：
- 资产收益 = 3
- 利息 = 2.5
- 净利润 = 0.5
- ROE = 0.5 / 50 = 1%（远低于无杠杆时的 3%）

如果 ROA 变成 2%：
- 资产收益 = 2
- 利息 = 2.5
- 净利润 = -0.5 → **杠杆吃掉了本金**

**结论**：杠杆是双刃剑。**顺周期加杠杆能加速财富积累，逆周期加杠杆会加速破产。**

### 3. 利息保障倍数（Interest Coverage Ratio）

$$
Interest Coverage = \frac{EBIT}{Interest Expense}
$$

经验阈值：
- > 8：非常安全，杠杆使用不足，可以再加
- 4-8：健康
- 2-4：偏紧，需警惕
- 1-2：危险，一次经营下滑就可能违约
- < 1：EBIT 都覆盖不了利息，处于财务困境

## 二、MM 定理（Modigliani-Miller）与现实修正

### MM I（1958）：完美市场下，资本结构无关

假设：无税、无破产成本、无信息不对称、无交易成本 → 公司价值 = 经营现金流 / WACC，与债权 / 股权比例无关。

**结论**：在完美市场，加杠杆只是"用更便宜的钱换掉更贵的钱，但同时提高股权风险，两者抵消"。

### MM II（1963）：有税修正

加入企业所得税后，因为利息抵税，公司价值随杠杆上升：
$$
V_{levered} = V_{unlevered} + T \times D
$$

**推论**：如果只考虑税盾（Tax Shield），最优杠杆是 100% 债务。

### 权衡理论（Trade-off Theory）：加入破产成本

现实中，杠杆过高会导致：
- 破产直接成本（律师、清算费用）
- 破产间接成本（客户流失、员工离职、供应商停止授信、优质资产贱卖）
- 财务困境成本（Financial Distress Costs）

$$
V_{levered} = V_{unlevered} + PV(Tax Shield) - PV(Financial Distress Costs)
$$

**最优杠杆点**：税盾边际收益 = 财务困境边际成本处。这解释了为什么现实中没有公司真的 100% 债务。

### 优序融资理论（Pecking Order）：信息不对称

Myers & Majluf（1984）：管理层比外部投资人更了解公司价值。发行股权 = "我觉得股价被高估" → 投资人打折。因此：

**融资优先级**：内源资金（留存利润）> 债务 > 混合工具（可转债、优先股）> 股权

**推论**：观察公司融资顺序可以推测管理层对股价的看法。频繁发股权 = 可能被高估。

### 市场时机理论（Market Timing）：机会主义

Baker & Wurgler（2002）：管理层会根据市场时机选择融资方式——牛市发股权、熊市发债权。

**实证**：中国 A 股定增高峰往往在指数高位（2015、2020），债券发行高峰在利率低点。**成熟 CFO 会主动利用市场周期做资本结构调整**。

## 三、中国现实的修正

MM 系列理论都基于成熟资本市场假设，用到中国需要以下修正：

### 修正 1：利率非市场化的历史遗留

- 2019 年前，LPR 改革之前，中国银行贷款利率有隐性下限
- 国企 / 央企拿到低于市场的利率（政策性利率），民企拿到高于市场的利率
- **实操**：算 WACC 时，民企的 R_d 要用**真实拿到的利率**，不能用 LPR 一刀切

### 修正 2：股权融资的窗口窄

- A 股 IPO 有窗口期（政策松紧、市场情绪）
- 再融资（定增、可转债）需证监会批准，节奏受管制
- 港股 / 美股相对市场化，但对内地民企有政治 / 监管风险溢价
- **实操**：不能假设"什么时候都能融到股权"，融资窗口本身是稀缺资源

### 修正 3：破产成本远高于教科书

- 中国破产法执行不畅，破产清算时间长（3-5 年）、回收率低（10-30%）
- 破产污名严重，创始人未来创业几乎不可能
- 供应商 / 员工 / 客户在"财务困境"信号出现后**极端敏感**（比欧美企业更敏感）
- **实操**：中国民企的最优杠杆点应显著低于教科书建议

### 修正 4：控制权溢价与股权稀释红线

- 家族企业老板往往有"51% 底线"或"绝对控股"心结
- 上市公司控制权变更触发要约收购（30% 红线）
- VIE 架构公司的"控制权"通过 AB 股或协议控制实现，稀释逻辑不同
- **实操**：股权融资方案必须显式列出稀释表和控制权变化

### 修正 5：明股实债 / 表外融资

- 中国存在大量"明股实债"（表面股权，实际有回购承诺）
- 融资租赁、保理、供应链金融、房地产的股权基金投资都可能是"表外债务"
- **实操**：算真实杠杆时要把这些还原到表内

## 四、行业典型资本结构

不同行业的最优资本结构相差极大：

### 重资产制造（钢铁、水泥、化工、汽车）
- D/E 典型区间：0.8 - 1.5
- 特点：固定资产可抵押、现金流相对稳定、需持续 Capex
- 风险：周期性强，衰退期利息保障骤降
- 案例：宝钢、海螺水泥、万华化学的 D/E 长期在 0.5-1.2

### 轻资产 SaaS / 互联网
- D/E 典型区间：0 - 0.3
- 特点：无形资产为主（不好抵押）、烧钱换增长、股权融资为主
- 风险：融资环境恶化时无法续命（2022-2023 美股 SaaS 集体重估）
- 案例：Salesforce D/E ~0.3、Shopify 早期几乎 0

### 房地产
- D/E 典型区间：2 - 5（历史上极端案例甚至 10+）
- 特点：项目融资、销售回款还债的现金流循环
- 风险：政策 / 销售 / 利率任何一个环节出问题就是雷（恒大、融创、碧桂园的教训）
- 案例：香港房企历史上 D/E 较低（1-2），内地房企因高杠杆模式已系统性调整

### 消费品
- D/E 典型区间：0.2 - 0.6
- 特点：品牌资产、现金流稳定、渠道占款
- 案例：农夫山泉 D/E < 0.3（现金流强到不需要杠杆）、名创优品类似

### 生物医药
- D/E 典型区间：0 - 0.5（研发期）；0.5 - 1.0（商业化后）
- 特点：研发期烧钱、不确定性高、股权为主；产品上市后可加杠杆
- 案例：百济神州融资以股权 + 可转债为主

### 金融 / 银行
- D/E 典型区间：8 - 15（银行）；2 - 5（保险 / 券商）
- 特点：本质就是"用别人的钱赚钱"，监管强制资本充足率
- 不适用一般企业逻辑

## 五、家族企业 / 上市公司 / 央国企的杠杆偏好

### 家族民企
- 偏好：**保守低杠杆**（老板不想欠钱），或**激进高杠杆**（老板赌性大）
- 分化明显，无统一模式
- 决策要点：不要用财务教科书说服老板，先理解其风险偏好
- 案例：老干妈几乎 0 债务（陶华碧"不贷款不上市不融资"）vs 部分二代扩张型民企杠杆 3+

### 上市公司
- 偏好：市值管理导向 —— 牛市增发（股权稀释可接受，因股价高）、熊市回购 + 加杠杆
- 决策要点：董事会 + 独立董事 + 大股东 + 机构股东多方博弈
- 陷阱：跟风分红 / 回购导致杠杆过高，一旦经营下滑就危险
- 案例：中国石化 / 中国神华的高分红传统 + 稳健杠杆

### 央国企
- 偏好：**利率优势明显**（政策性利率、政府担保），但受**资产负债率红线**约束（国资委对央企有考核）
- 决策要点：资本结构决策带有政治任务色彩（如"降杠杆"是过去几年硬指标）
- 案例：中国建筑、中国铁建等在"降杠杆"考核下从 D/E 4+ 降到 2 左右

### 独角兽 / Pre-IPO 高成长企业
- 偏好：几乎全股权（VC 轮融资）
- 特殊工具：可转债、优先股、优先清算权（Preference）
- 决策要点：估值 + 稀释 + Preference 条款的组合权衡

## 六、中间层融资工具

除了纯债 + 纯股，现代 CFO 有更多工具箱：

### 可转债（Convertible Bond）
- 结构：债券 + 转股期权
- 适用：估值不确定 / 短期不想稀释 / 需要过渡期
- 中国案例：A 股上市公司可转债市场发达（金融机构、消费、制造业多有使用）
- 风险：转股条款设计不当会导致大规模稀释或提前赎回压力

### 优先股（Preferred Stock）
- 结构：固定股息 + 优先清算权，无投票权（或有限投票权）
- 适用：VC/PE 融资、家族企业股权改革
- 中国限制：A 股上市公司可发优先股（工行、农行、浦发等银行大量使用），非上市公司范围有限

### 永续债（Perpetual Bond）
- 结构：无固定期限的债，会计上可能计入"权益"
- 适用：想加杠杆但不影响 D/E 报表数字（央企常用）
- 陷阱：投资人视其为"变相股权 + 高息"，实际成本高

### ABS / REITs（资产证券化）
- 结构：把特定资产（应收、租金、门店）打包出售
- 适用：轻资产化、盘活存量、表外融资
- 中国案例：万科 / 华润的 CMBS、公募 REITs（2021 年后开闸）

### 战略投资人（Strategic Investor）
- 结构：股权 + 业务协同承诺
- 适用：需要产业资源 + 资本
- 案例：腾讯、阿里、字节的 CVC 投资遍布消费 / 教育 / 电商赛道
- 权衡：拿了 A 家的钱可能就拿不了 B 家（站队问题）

## 七、给 strategy-cfo 的用法建议

- 每次战略决策必给"资本结构建议"表：融资组合 + WACC 影响 + 杠杆比率 + 利息保障 + 触发降杠杆条件。
- **不要为"低 WACC"而加杠杆**：低 WACC 不是目的，价值最大化才是。加杠杆同时会提高股权成本（β 上升）。
- **顺周期与逆周期区别对待**：宏观景气 + 现金流强 = 可以加杠杆；宏观下行 + 现金流承压 = 主动去杠杆。
- **家族 / 央国企 / 上市三种客户的资本结构建议要"文化敏感"**：不要用一套模板。
- **警惕中国式表外债务**：明股实债、供应链金融、融资租赁都要还原到真实杠杆里。
- **利息保障倍数是最重要的红线**：低于 3 就要预警，低于 2 是危险，低于 1 是紧急。
- **重大战略决策的资本结构必须给至少 2 种方案**（保守 vs 进取），列出各自的股权稀释、控制权变化、财务风险、后续融资空间。
