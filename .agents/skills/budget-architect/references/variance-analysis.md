# 差异分析

差异分析（Variance Analysis）是预算管理的"神经系统"：如果预算是骨架，差异分析就是让骨架能感知冷热痛痒的神经。做得对，管理层能提前 1-3 个月发现问题；做得错，就是月末摆样子的表格。

## 1. 基本框架：什么是差异

**差异 = Actual - Budget**（或 vs Forecast）

**基本判定**：

- 收入/毛利/利润类：Actual > Budget → **Favorable (F)** 有利差异
- 成本/费用/亏损类：Actual < Budget → **Favorable (F)** 有利差异

反之为 Unfavorable (U) 不利差异。

**警告**：不能只看单期差异。差异分析必须结合"趋势 + 累计 + 同期"。

## 2. 三大差异分解（收入侧）

以收入为例，差异可分解为：

**价差 (Price Variance)** + **量差 (Volume Variance)** + **混合差 (Mix Variance)** = **总差异**

### 2.1 单产品的价差量差

设：
- 预算量 = Qb，预算价 = Pb
- 实际量 = Qa，实际价 = Pa

则：
- 价差 = (Pa - Pb) × Qa
- 量差 = (Qa - Qb) × Pb
- 总差异 = Pa × Qa - Pb × Qb = 价差 + 量差

**worked example**：

某产品预算：卖 10000 件 × 100 元 = 100 万
实际：卖 12000 件 × 95 元 = 114 万
总差异 = 114 - 100 = +14 万（Favorable）

- 价差 = (95 - 100) × 12000 = -6 万（Unfavorable，降价 5 元）
- 量差 = (12000 - 10000) × 100 = +20 万（Favorable，多卖 2000 件）

**解读**：靠降价拉动了销量，最终有利，但要警惕：
- 是主动降价（战略选择）还是被动降价（竞争加剧）？
- 降价 5% 换销量 20% 是否可持续？
- 毛利总额是否也有利？

### 2.2 多产品的混合差异（Mix Variance）

当有多个产品线时，混合差异捕捉"高价产品占比变化"：

- 预算总量 = 10000（A 60% + B 40%），预算单价 A=200, B=100，预算总收入 = 6000×200 + 4000×100 = 160 万
- 实际总量 = 10000（A 50% + B 50%），实际单价 A=200, B=100（假设未变），实际总收入 = 5000×200 + 5000×100 = 150 万
- 差异 = -10 万（Unfavorable）

尽管总量和单价都没变，但因为 A（高价品）占比下降，整体收入下降。这就是纯混合差异。

**分解公式**：`混合差异 = Σ (实际占比 - 预算占比) × 预算单价 × 实际总量`

## 3. 三大差异分解（成本侧）

### 3.1 直接材料成本

- 材料价差 = (Pa - Pb) × Qa（消耗量按实际算）
- 材料量差 = (Qa - Qb) × Pb（价格按预算算）

**worked example**：

某产品预算材料成本：1 件用 2kg × 50 元/kg = 100 元
实际：1 件用 2.2kg × 55 元/kg = 121 元

- 材料价差 = (55 - 50) × 2.2 = +11 元（Unfavorable，涨价了）
- 材料量差 = (2.2 - 2) × 50 = +10 元（Unfavorable，多用了）

总材料成本超支 21 元/件，其中 11 元来自涨价、10 元来自浪费。

### 3.2 直接人工成本

- 人工价差 = (工资率实际 - 工资率预算) × 实际工时
- 人工效率差 = (实际工时 - 预算工时) × 工资率预算

### 3.3 制造费用（Overhead）

变动制造费用 + 固定制造费用需要分别处理。

## 4. 差异根因诊断

**差异不是终点，根因才是**。以下是常见差异的根因诊断路径：

### 4.1 收入不利（Actual < Budget）

- 量差为主：
  - 客户流失？ → 市场部拉客户流失清单，销售部一对一复盘
  - 竞对压价？ → 商务信息、市占率数据
  - 产品问题？ → 售后投诉、退货率
  - 宏观？ → 行业数据、GDP、消费指数
- 价差为主：
  - 主动降价（促销）？ → 检查是否与营销策略一致
  - 被动降价（竞争）？ → 竞对报价监控
  - 客户结构变化（大客户占比升，议价压低）？ → 客户分级 P&L
- 混合差异为主：
  - 高价品下滑？ → 产品线组合策略回顾

### 4.2 毛利率下降

- 收入结构变化（低毛利品占比升）
- 原材料涨价未传导
- 生产效率下降（工时超支、废品率上升）
- 分销渠道结构变化（直销转分销）
- 库存跌价准备（滞销品堆积）

### 4.3 期间费用超支

- 销售费用：营销活动 ROI 不及预期？销售提成计算方式变化？差旅超支？
- 管理费用：招聘超编？加薪超预算？咨询费临时增加？
- 研发费用：项目延期、人力外包成本超支？

## 5. 预警阈值设计

### 5.1 三色预警框架

| 指标 | 绿灯 | 黄灯 | 红灯 | 触发动作 |
|---|---|---|---|---|
| 营收累计进度 | ±5% | -5% ~ -10% | <-10% | 黄：销售总监周会说明；红：CEO 一对一 |
| 毛利率 | ±1pp | -1 ~ -3pp | <-3pp | 黄：产品部检讨；红：全面暂停促销 |
| 期间费用累计 | ±5% | +5% ~ +10% | >+10% | 黄：冻结非核心招聘；红：CFO 逐笔审批 |
| 经营性现金流 | ±10% | -10% ~ -20% | <-20% | 黄：应收清收专项；红：暂停 Capex |
| 单月净利 | ±10% | ±10-20% | ±>20% | 黄：管理层运营会；红：董事会通报 |

### 5.2 阈值分级原则

- **绝对金额 + 相对比例双门槛**：不能只看比例（小数额 100% 差异不重要），也不能只看绝对（大公司 5% 差异可能已经很严重）
- **累计 vs 单月**：允许单月大幅波动，但累计不能持续偏离
- **不同科目不同标准**：营收、毛利、费用、现金流的阈值应该差异化设置

### 5.3 阈值调整节奏

- 阈值不是一次性定死，每季度或每年 review 一次
- 业务波动加剧时（如疫情、大周期变化），阈值应放宽
- 业务稳定后，阈值应收紧（避免麻木）

## 6. 弹性预算 vs 静态预算

### 6.1 静态预算的问题

传统预算是"静态"的：卖 100 万件按 100 万件的成本预算。但实际卖 120 万件时，成本"当然"会更高——不能简单对比。

### 6.2 弹性预算（Flexible Budget）

按实际业务量重新计算的成本预算。

**公式**：`弹性预算 = 固定成本 + 单位可变成本 × 实际业务量`

**worked example**：

预算：卖 100 万件，可变成本 60 元/件（6000 万），固定成本 2000 万，总成本 8000 万。
实际：卖 120 万件，实际总成本 8500 万。

- 静态差异 = 8500 - 8000 = -500 万（Unfavorable，看起来超支）
- 弹性预算 = 2000 + 60 × 120 = 9200 万
- 弹性差异 = 8500 - 9200 = +700 万（Favorable！实际是节约了）
- 业务量差异 = 9200 - 8000 = -1200 万（这是"因为卖更多所以成本更多"，属于正常）

**结论**：静态预算看起来超支 500 万，实际经营效率是提升的（节约 700 万），只是被"卖更多本来就要花更多"掩盖了。

**关键洞察**：任何成本超支分析，必须先做弹性预算调整，否则会冤枉运营团队或纵容浪费。

## 7. 月度差异分析报告模板

```markdown
# XX 月经营差异分析报告

## 一、执行概览
- 营收：Actual 1.05亿 vs Budget 1.00亿 → +5.0% F
- 毛利：Actual 3200万 vs Budget 3500万 → -8.6% U
- 经营利润：Actual 800万 vs Budget 1200万 → -33.3% U
- 现金流：Actual 500万 vs Budget 800万 → -37.5% U

**一句话判断**：收入超预期但毛利承压、费用超支，现金流亮红灯。

## 二、Top 3 差异
### 1. 毛利率下降 3.4pp
- 价差：-500 万（主动降价 3% 抢单）
- 材料成本超支：+300 万（钢材涨价 8%）
- 影响：本月毛利少 800 万，占经营利润差异的 80%
- 根因：Q1 客户还价激烈 + 供应商未锁长单

### 2. 销售费用超支 15%
- 展会+营销 +200 万（新签客户 5 家，客户 LTV 待验证）
- 商务差旅 +80 万（Q1 出海拓客）

### 3. 应收账款周转天数从 45 天 → 62 天
- 3 家大客户账期主动延长
- 影响经营性现金流 -400 万

## 三、根因分析
### 毛利承压
- 直接原因：钢材涨价 + 主动降价
- 深层原因：Q1 淡季客户议价能力强，长单锁价机制未建立

### 现金流恶化
- 直接原因：大客户账期延长
- 深层原因：给大客户的授信管理没有跟上业务节奏

## 四、下月行动
- [ ] 供应链部：4 月底前完成钢材年度长单签约（目标锁价 -3%）
- [ ] 销售部：Top 5 客户账期回归 45 天，逾期客户开通催收
- [ ] 财务部：4 月中弹性预算重算，5 月启动 Q2 滚动更新

## 五、预警
- 黄灯：毛利率累计低于预算 2pp（连续 2 月）
- 红灯：经营性现金流累计低于预算 25%

## 六、下季度关注
- 大客户账期是否稳定回归
- 钢材长单能否落地
- 新签客户 LTV 验证
```

## 8. 差异分析常见坑

1. **只看金额不看趋势**：单月差异可能是季节性，累计和同比才有意义
2. **不做弹性预算**：把销量增长带来的成本上升归咎为"超支"
3. **只讲现象不讲根因**：报告写成"营销费超支 100 万"就没了，不追根因永远解决不了
4. **差异全归财务**：财务只能算差异，根因诊断必须业务部门参与
5. **预警阈值失效**：黄灯亮了没人管，红灯亮了才慌 → 必须建立触发即行动的机制
6. **只做 P&L 差异**：忽略资产负债表和现金流量表差异（应收、存货、Capex 差异往往更致命）
7. **差异分析报告写给财务看**：写得都是"应付账款周转天数 X 天"业务看不懂 → 必须翻译成业务语言（"我们比预算多欠供应商 XX 天"）
8. **月月复盘同一个差异**：说明根因诊断没做到位或行动方案没执行 → 复盘节奏要收敛

## 9. 差异分析成熟度

- **Level 1**：月末财务算出 Actual vs Budget 差异表 → 大部分公司停在这里
- **Level 2**：加入价差量差分解 + 根因分析 + 行动清单
- **Level 3**：弹性预算 + 三色预警 + 触发动作 + 部门责任人绑定
- **Level 4**：实时看板（Power BI / Tableau），单日单周即可看差异
- **Level 5**：预警前置（用 leading indicator 预测差异，而不是等 Actual 出来）
