# 估值视角（DCF / 可比公司 / 可比交易 / 资产法）

估值是战略决策的核心量尺。所有资本配置、融资、并购、IPO、退出决策，最终都要回答一个问题：**这个东西值多少钱？** strategy-cfo 用四大主流估值方法（现金流折现、可比公司、可比交易、资产法）为战略决策提供估值区间，同时警惕估值陷阱。

## 一、四大估值方法总览

| 方法 | 逻辑 | 适用场景 | 主要缺点 |
|---|---|---|---|
| **DCF（现金流折现）** | 内在价值 = 未来现金流的现值 | 现金流可预测的成熟企业 / 单个项目 | 对假设极敏感、终值主导 |
| **可比公司（Trading Multiples）** | 用同行的估值倍数横向类推 | 有充分可比公司的成熟行业 | 市场情绪影响、可比性差 |
| **可比交易（Transaction Multiples）** | 用近期并购交易倍数类推 | 并购定价 / 私有化 / 出售 | 控制权溢价、交易稀少 |
| **资产法** | 资产减负债的净值 | 重资产 / 清算场景 / 金融机构 | 忽略盈利能力和商誉 |

实操中往往**三角验证**：DCF + 可比公司 + 可比交易 三个方法给出的估值区间彼此对照，如果三个方法差异 > 30%，说明某个假设有问题。

## 二、DCF（Discounted Cash Flow）估值

### 1. 基本公式

$$
Enterprise\ Value = \sum_{t=1}^{n} \frac{FCFF_t}{(1+WACC)^t} + \frac{Terminal\ Value}{(1+WACC)^n}
$$

$$
Equity\ Value = Enterprise\ Value - Net\ Debt
$$

- FCFF = 自由现金流（Free Cash Flow to Firm）= EBIT × (1-T) + 折旧摊销 − Capex − 营运资本变动
- Terminal Value = FCFF_{n+1} / (WACC − g) 或 Exit Multiple × EBITDA_{n}

### 2. 关键假设及其影响

DCF 的所有争议都集中在几个核心假设：

**（a）预测期收入增长率**：一般显式预测 5-10 年
- 案例：高增长 SaaS 公司预测期 CAGR 30% vs 成熟消费品 5% → 估值差数倍

**（b）利润率轨迹**：起点利润率 → 稳态利润率
- SaaS 从早期负利润率 → 长期净利率 20-30%（"规模化后利润率大幅提升"是估值故事的核心）
- 陷阱：假设的稳态利润率高于行业最优，需要额外论证

**（c）Capex 与营运资本**：
- 稳态期 Capex ≈ 折旧摊销
- 增长期 Capex 显著高于折旧

**（d）WACC**：详见 `capital-structure.md`
- 常见陷阱：用了过时的历史 WACC（2020 加息前的低 WACC 用到 2024 会严重高估）

**（e）永续增长率（g）**：一般 2-4%
- **绝对红线**：g < WACC（否则数学上无意义 → 无穷大）
- **保守取值**：不超过长期 GDP 增速（3-5%）
- **陷阱**：g 从 3% 调到 4% 可能让估值上升 20-30%，稍不留神就"调出想要的答案"

### 3. 终值主导问题

DCF 估值中，终值（Terminal Value）通常占总估值的 **60-80%**。这意味着：
- 前 10 年现金流预测再精细，终值假设一变全都白搭
- **稳态假设是关键**：稳态时的利润率、增长率、Capex 需求

**验证方法**：
- 让终值时的隐含 EV/EBITDA 倍数与同行成熟公司比较（差异过大要重新审视）
- 让终值时的隐含 P/E 与市场长期均值比较

### 4. DCF 变体：DDM / FCFE / rNPV

- **DDM（股息折现模型）**：用于成熟高分红公司（银行、公用事业）
- **FCFE（股东自由现金流折现）**：直接算股权价值，不算 EV
- **rNPV（风险调整 NPV）**：每阶段乘以成功概率 —— 新药 / 硬科技 / 早期资源开发

## 三、可比公司法（Trading Multiples）

### 1. 常用倍数

| 倍数 | 分子 | 分母 | 适用 |
|---|---|---|---|
| **P/E** | 股价 | 每股盈利 | 有稳定盈利的公司 |
| **P/S** | 股价 | 每股营收 | 高增长 / 亏损公司 |
| **P/B** | 股价 | 每股净资产 | 金融 / 重资产 |
| **EV/EBITDA** | 企业价值 | EBITDA | 跨杠杆结构比较（M&A 常用） |
| **EV/Revenue** | 企业价值 | 营收 | 亏损 SaaS / 互联网 |
| **PEG** | P/E | 增长率 | 高增长公司（PEG < 1 常被认为便宜） |

### 2. 使用步骤

1. **选可比公司**：同行业 + 相近规模 + 相近增长阶段 + 相似业务模式
2. **计算各家倍数**：注意剔除极端值
3. **取中位数 / 均值**：中位数抗极端值，均值敏感
4. **应用到目标公司**：目标 EBITDA × 行业中位 EV/EBITDA = 估值

### 3. 中国实操要点

**（a）A 股 vs 港股 vs 美股倍数差异极大**：
- A 股同类公司往往有"稀缺性溢价"或"流动性溢价"
- 港股折价常见（"折价港"）
- 中概股受地缘政治影响，倍数经常处于折价
- **同一家公司**（如百济神州、理想汽车）在两地上市，倍数可能差 20-30%

**（b）可比公司数量陷阱**：
- 少于 5 家的可比样本不可靠
- 中国特色行业（如白酒、军工、稀土）可比对象只在 A 股 —— 与全球估值脱钩

**（c）时点问题**：
- 用过去 6-12 月的平均倍数 vs 当前时点倍数会给出不同答案
- 熊市时用 12 月均值可能高估、牛市用当前时点可能高估

### 4. 常见倍数陷阱

- **用周期顶点的 P/E**：钢铁、化工、有色的 P/E 在周期顶点看起来极便宜（3-5x），但那是"顶点收益 / 高股价"，很快回归
- **PEG 陷阱**：增长率用未来 1 年、还是 3-5 年 CAGR，答案差很多
- **EV/EBITDA 忽略 Capex**：EBITDA 相同但 Capex 需求不同的公司，实际现金流大不同
- **中国式财报口径差异**：一次性收益 / 政府补贴 / 投资收益是否剔除，倍数会大差

## 四、可比交易法（Transaction Multiples）

### 1. 逻辑

不是"目前市场愿意付什么价"（Trading），而是"实际交易时买家付了什么价"（Transaction）。

$$
Deal\ Value / Target\ EBITDA = Transaction\ Multiple
$$

### 2. 与可比公司的差异

| 维度 | 可比公司（Trading） | 可比交易（Transaction） |
|---|---|---|
| 数据来源 | 上市公司公开股价 | 并购公告 / IB 数据库 |
| 反映的价值 | 少数股东权益价值（Minority） | 控制权价值（Control） |
| 溢价含义 | 无控制权溢价 | **含 20-30% 控制权溢价** |
| 适用 | 判断"上市值多少" | 判断"卖公司值多少" / "买公司要出多少" |

### 3. 控制权溢价（Control Premium）

买 100% 股权（获得控制权）比买 1% 股权（无控制权）单价高。中国典型控制权溢价 **20-30%**，因人而异：
- 战略买家（同业 / 上下游）愿意付更高溢价（协同效应）
- 财务买家（PE）付溢价较低（无经营协同）
- 上市公司要约收购溢价 = 前 30 日均价 × (1 + 15%~25%)

### 4. 中国 M&A 交易数据来源

- Wind / Choice / iFind 数据库（收费）
- 证监会 / 深沪港交易所公告（免费但需自己整理）
- 商务部反垄断公告
- 头豹 / 艾瑞 / CVSource 等私募数据
- **中国跨境 M&A**：另参考 Refinitiv / Bloomberg 数据

### 5. 陷阱

- 交易数量少 → 少数极端交易主导估值
- 时点集中（如 2015-2016 A 股跨境并购潮）→ 溢价过高
- 支付方式不同（现金 vs 股票 vs 混合）会影响观察到的成交价
- 对赌条款、earn-out 会让真实成交价与公告价有差异

## 五、资产法（Asset-Based Approach）

### 1. 三种子方法

**（a）账面价值（Book Value）**：直接看资产负债表净资产
- 优点：简单
- 缺点：账面价值往往严重低估（无形资产、土地增值、品牌价值都未反映）

**（b）重置成本（Replacement Cost）**：重建同样资产要多少钱
- 用于：制造业、基础设施
- 缺点：忽略盈利能力（有些资产明明能赚钱，重置成本反而低）

**（c）清算价值（Liquidation Value）**：破产时能卖多少
- 用于：财务困境公司、投资保底、清算分析
- 中国实操：破产清算回收率往往 10-30%（远低于账面）

### 2. 适用场景

- 重资产行业（房地产、矿业、船运、金融）
- 破产 / 清算 / 严重财务困境
- 控股型企业估值（Sum of Parts 里的某个业务）
- 保底估值（估值区间的下限）

### 3. 中国式资产估值特殊性

- 土地使用权：账面成本远低于市价，尤其一二线城市 —— 老国企 / 老民企的"隐藏资产"
- 商标 / 专利 / 数据：会计上多计入无形资产但估价严重不足
- 股权投资：账面按成本法或权益法计量，市值可能高数倍

## 六、SOTP（Sum-of-the-Parts）分部估值

多业务集团整体估值 = 各业务分别估值之和。

### 使用场景

- 综合集团（如伯克希尔、腾讯、阿里）
- 上市公司分拆前后估值分析
- 战略重组决策

### 步骤

1. 拆解业务线（按现金流可辨认性）
2. 每条业务线用最合适的方法估值（成熟业务 DCF、SaaS 用 P/S、金融用 P/B、房地产用 NAV）
3. 加总各业务估值
4. 减去总部成本 / 加上非经营性资产（如现金、投资）
5. 应用**集团折价**（Conglomerate Discount，通常 15-25%）

### 集团折价 vs 集团溢价

**集团折价来源**：
- 交叉补贴、资本配置低效
- 管理复杂性
- 分析师难以覆盖多个赛道

**集团溢价来源**：
- 内部协同（渠道 / 供应链 / 数据）
- 内部资本市场（现金牛反哺明星）
- 抗周期能力

**中国现实**：家族多元化集团典型集团折价 20-30%（甚至更高）；巨头（腾讯 / 阿里 / 平安）折价程度看市场情绪。

### 案例：腾讯 SOTP

- 游戏业务：DCF 或 P/E
- 广告业务：P/S
- 金融科技 / 云：P/S
- 投资组合：市值加总（Sea、Meituan、Kuaishou、Pinduoduo、Tesla 等）
- SOTP 后 → 应用集团折价（历史区间 10-25%）

## 七、战略决策如何影响估值

估值不是被动测算，是**战略决策的因变量**。CFO 视角看，战略决策通过以下路径影响估值：

### 路径 1：改变收入增长故事
- 进入高增长市场 → 未来 5 年 CAGR 从 10% → 25% → DCF 估值 + 50%
- 陷阱：新市场增长故事被证伪 → 估值可能回落到起点甚至更低（增加了不确定性）

### 路径 2：改变利润率
- 转型 SaaS → 稳态净利率从 8% → 20% → 估值倍数从 P/E 15x → P/S 8x（可能是 3-5 倍差异）
- 陷阱：转型期利润率短期恶化（Adobe 2013 教训）

### 路径 3：改变估值倍数
- 从"传统制造"重估为"新能源 / AI 概念" → 估值倍数从 15x P/E 跳到 40x P/E
- 陷阱：故事无法兑现时倍数回归极快

### 路径 4：并购溢价 vs 稀释
- 并购溢价 30% → 若协同兑现，NPV > 0；协同不足则商誉减值
- 股权融资稀释 20% → 每股价值下降，除非融资加速带来 20%+ 的估值提升

### 路径 5：控制权与流动性
- 上市 → 提高流动性 → 估值溢价 20-40%
- 拆分上市（分拆子公司）→ 消除集团折价
- 私有化 → 承担控制权溢价（20-30%）

### 路径 6：ESG / 治理评级
- 治理改善 / 独董比例 / 环境表现 → 机构评级上调 → 估值倍数微调（3-5%）
- 中国国资背景公司近年 ESG 权重上升

## 八、一级市场 vs 二级市场估值方法差异

**极重要**：同一家公司在一级（未上市）和二级（上市）市场往往估值差别巨大。原因：

### 一级市场（VC / PE / 私募）
- 主要方法：可比交易 + 讨价还价 + 战略溢价 + 稀释谈判
- 估值来源：故事 + 领投人认可 + 增长率
- 时间视野：3-7 年（退出周期）
- 流动性：极低（几乎不能中途退出）
- **典型倍数**：SaaS 一级市场 P/S 15-30x（2020-2021 高峰）→ 2023-2024 回落到 5-10x

### 二级市场（公开市场）
- 主要方法：DCF + 可比公司 + 交易情绪
- 估值来源：季报节奏 + 分析师评级 + 宏观利率 + 板块轮动
- 时间视野：季度到年度
- 流动性：高
- **典型倍数**：SaaS 二级市场 P/S 加息周期 3-8x

### 一二级估值倒挂
2022 年后大量独角兽发现："一级市场 D 轮估值 = 300 亿，如果今天上市二级市场只给 100 亿"。这就是估值倒挂。

**原因**：
- 一级市场 2020-2021 追高
- 二级市场对利率、增长可持续性打折
- 私募投资人的优先清算权保护了他们，但普通股价值受挤压

**strategy-cfo 处理估值倒挂建议**：
- 避免用一级估值反推二级（一级市场逻辑不同）
- 一级估值高但短期无退出通道 = 纸上富贵
- 融资时慎选估值 —— 高估值 + 优先清算权 + 反稀释保护 = 未来卖公司或 IPO 时创始人可能一无所有

## 九、常见估值陷阱

### 陷阱 1：永续增长率 > 长期 GDP 增速
- g = 5% 或以上 → 数学上是可以，但经济上意味着"这家公司未来永远比 GDP 快"→ 不合理

### 陷阱 2：把周期顶点当基线
- 周期性行业（钢铁、化工、有色、地产、大宗）估值时用顶点年份为基线 → 估值系统性高估
- **正确做法**：用 3-5 年周期均值

### 陷阱 3：忽略稀释
- 期权池、可转债、可执行认股权（Warrants）在算 P/E 时应算摊薄后（Diluted）
- 融资方案中的股权 dilution 是"未来的隐性成本"

### 陷阱 4：协同溢价被过度支付
- 并购普遍高估协同效应 50%+，低估整合成本 30%+
- **保守做法**：协同溢价打 50% 折

### 陷阱 5：控制权溢价反向使用
- 卖方视角：控制权溢价是"你要额外付的钱"
- 买方视角：控制权溢价必须匹配"你能实际获得的额外价值"
- 陷阱：付了 30% 控制权溢价但整合失败 = 溢价打水漂

### 陷阱 6：只看倍数不看规模
- P/E 20x 的两家公司：一家 EPS 1 元、一家 EPS 100 元 → 前者可能只是小盘股情绪推高、后者是大盘蓝筹稳定 —— 风险不同

### 陷阱 7：忽略非经营性资产 / 负债
- 高额现金、股权投资、诉讼、担保、表外债务
- SOTP 时特别重要 —— 腾讯的投资组合就占相当估值

### 陷阱 8：高增长 vs 稳态期估值 lens 差异

**高增长期**：估值倍数（P/S、EV/Revenue）高，重视增长率、市占率、GMV、DAU
**稳态期**：估值倍数（P/E、EV/EBITDA）常态，重视利润率、现金流、分红

**陷阱**：转型时 lens 切换不当
- 亚马逊长期用 P/S / 现金流估值 → 2020 后开始用 P/E
- 阿里 / 腾讯从"增长股"lens 逐步转向"价值股"lens
- 陷阱：市场比公司自己更早切换 lens → 股价先跌 → 公司被动

## 十、战略正确 vs 财务合理的冲突

CFO 视角下最常见的两难：

### 情境 1：战略必要但 NPV 负
- 例：卡位新赛道、防御性并购、买断某个专利
- 处理：显式量化"战略溢价"（如"如果不做，未来 3 年市场份额可能被抢 X%"）
- 决策：董事会明确接受战略溢价 → 通过；无法量化 → 需重新评估

### 情境 2：财务漂亮但战略偏离
- 例：某个高 IRR 项目但不属于主航道
- 处理：一次性机会主义投资是可以的（占资本预算 < 10%）
- 陷阱：连续多个"财务漂亮但偏航"的项目 → 战略漂移

### 情境 3：短期财务恶化换长期战略
- 例：SaaS 转型、平台化、砍掉现金牛业务
- 处理：给二级市场一个清晰的"J 曲线"故事 + 关键 gate check
- 陷阱：市场耐心有限 —— 通常给 6-12 个月，超过就会用脚投票

### 情境 4：财务不错但估值故事讲不出来
- 例：稳定分红的成熟制造企业
- 处理：接受"低估值 + 高分红"的价值股定位，不硬转型
- 陷阱：管理层试图讲增长故事但市场不买单 → 估值不涨反跌（讲了做不到）

## 十一、给 strategy-cfo 的用法建议

- **每次估值给区间不给单点**：DCF + 可比公司 + 可比交易的三角区间 → 落到区间中枢 ± 20% 是合理表述。
- **主导方法要匹配业务性质**：SaaS / 亏损公司用 P/S；成熟企业用 DCF + P/E；重资产用 EV/EBITDA + 资产法；多业务用 SOTP。
- **敏感性必做**：WACC、g、利润率至少三个变量的敏感性分析。
- **中国市场警觉性**：地缘政治、监管、税收优惠退出、板块轮动都能让估值倍数骤变。
- **一二级估值必区分**：不要交叉引用。
- **战略决策对估值的影响必量化**：DCF 里做"当前基线 vs 实施战略后"两个模型，差异即战略价值。
- **战略溢价必显式**：任何 NPV 负但推荐做的项目，"战略溢价"具体是多少 / 什么条件下不成立要写清楚。
- **不做正式估值报告**：strategy-cfo 只给建模思路和估值区间；一二级市场融资 / 并购 / IPO 的正式估值需 IB / FA / 独立审计师加持。
- **每年更新 WACC**：加息 / 降息周期 / 行业风险溢价变化 → WACC 至少每年更新一次。
