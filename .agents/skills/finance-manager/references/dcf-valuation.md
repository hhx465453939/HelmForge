# DCF 估值（Discounted Cash Flow）

DCF 是把一家公司未来能产生的自由现金流，用一个反映风险的折现率贴现回今天。它的核心公式简单，但每一个参数都能藏坑，DCF 的实际结果对参数极其敏感。

## 1. 核心公式

    企业价值（EV）= Σ FCFF_t / (1 + WACC)^t  +  终值 / (1 + WACC)^N
    股东权益价值 = EV - 有息净债务
    每股价值 = 股东权益价值 / 总股本

或用 FCFE（对股东的自由现金流）直接对股东权益估值：

    股东权益价值 = Σ FCFE_t / (1 + Ke)^t + 终值 / (1 + Ke)^N

两种方法只要保持"分子对应分母"的一致性即可（FCFF 配 WACC，FCFE 配 Ke）。

## 2. FCFF / FCFE 的计算

### 2.1 FCFF（Free Cash Flow to Firm）

    FCFF = EBIT × (1 - 有效税率)
         + 折旧摊销
         - 资本开支（CapEx）
         - 营运资本增加（ΔWC）

其中：
- **EBIT × (1 - 税率)** = NOPAT（无杠杆的税后经营利润）
- **折旧摊销**：非现金费用要加回
- **资本开支**：维持性 CapEx + 扩张性 CapEx，都要扣
- **营运资本变化**：(应收 + 存货 - 应付) 的净增加要扣

### 2.2 FCFE（Free Cash Flow to Equity）

    FCFE = 净利润 + 折旧摊销 - CapEx - ΔWC + 净借款

或

    FCFE = FCFF - 利息(1 - 税率) + 净借款

**注意**：FCFE 对资本结构变化敏感，稳定性差；FCFF 更常用。

## 3. WACC 与 Ke 估算

### 3.1 WACC（加权平均资本成本）

    WACC = (E / (D + E)) × Ke + (D / (D + E)) × Kd × (1 - 税率)

其中：
- E = 股东权益市值（不是账面）
- D = 有息负债市值（近似账面）
- Ke = 权益资本成本
- Kd = 债务成本（一般用现有借款利率或同评级公司债收益率）

### 3.2 Ke 用 CAPM

    Ke = 无风险利率 + β × 市场风险溢价

**中国参考取值（2024-2025 大致水平）**：
- 无风险利率：10 年期国债收益率 ≈ 2.3-2.8%
- 市场风险溢价（MRP）：6-8%（各机构口径不同）
- β：从可比公司 5 年月度回归得到，或用行业 β 平均

**美国参考取值**：
- 无风险利率：10 年期美债收益率 ≈ 4-4.5%
- MRP：4.5-6%
- β：Damodaran 网站行业 β 表

### 3.3 β 的注意事项

- 用可比公司 β 时先"卸载财务杠杆"再按目标公司结构"重新加载"：

        β_unlevered = β_levered / (1 + (1 - t) × D/E)
        β_relevered = β_unlevered × (1 + (1 - t) × D/E)

- 单一公司 β 波动大时，取行业平均更稳健

## 4. 显性预测期 + 终值

### 4.1 显性预测期

- **一般 5-10 年**（成熟公司 5 年，高成长公司 7-10 年）
- 显性期越长，越依赖长期增长假设的准确性
- 显性期结束时，公司应处于"稳态"（增长收敛到永续水平，投资回报率接近 WACC）

### 4.2 终值（Terminal Value）

两种主流方法：

**A. 永续增长法（Gordon Growth）**

    TV = FCFF_(N+1) / (WACC - g)
       = FCFF_N × (1 + g) / (WACC - g)

- g = 永续增长率
- **关键约束**：g ≤ 长期名义 GDP 增速（中国 ≈ 3-5%，美国 ≈ 2-3%）
- **常见错误**：g 拍到 5-8%，其实等于假设公司永远比 GDP 长得快，这是数学上不成立的

**B. 退出乘数法（Exit Multiple）**

    TV = EBITDA_N × 退出 EV/EBITDA 倍数

- 用行业当前的稳态倍数（历史中位数或成熟公司平均）
- 优点：贴近市场；缺点：将相对估值风险嵌入 DCF

**实操建议**：两种方法都算一次，交叉验证。差异过大时找参数问题。

## 5. 敏感性分析（必做）

DCF 结果对**WACC 和 g（或退出倍数）** 极其敏感。建议输出 2 维敏感表：

|          | g=2% | g=2.5% | g=3% | g=3.5% | g=4% |
|----------|------|--------|------|--------|------|
| WACC=8%  |      |        |      |        |      |
| WACC=9%  |      |        |      |        |      |
| WACC=10% |      |        |      |        |      |
| WACC=11% |      |        |      |        |      |

**结论表达**：不要只给一个"公允价值 = ¥XX"，而是给出"在合理参数区间内，公允价值 ¥XX - ¥YY"。

## 6. 常见陷阱

1. **永续增长率高于长期 GDP 增速**：违反数学，公司不能永远快于经济
2. **WACC 拍脑袋**：随便取 10% 而不做 CAPM 推导
3. **忽略资本开支**：只加折旧、不减 CapEx，虚增 FCFF
4. **营运资本忽略**：高增长公司 ΔWC 常常吃掉大量现金
5. **税率用名义税率而非有效税率**：有效税率往往因优惠远低于 25%
6. **一次性收入 / 支出未剔除**：显性期基数错，后面全错
7. **循环引用**：Ke 用市值权重，市值又靠 DCF 算——需要迭代 or 用目标资本结构
8. **少数股东权益处理**：合并报表 FCFF 里包含子公司现金流，最后减 EV 时应减少数股东权益
9. **持有现金处理**：EV 减净有息债务时，"多余现金"要单独加回
10. **中期年份贴现**：现金流通常年中产生，用 mid-year convention 贴现更准

## 7. DCF 不适用的场景与替代方法

DCF 有明确的适用边界。以下场景 DCF 效果差或不适用：

### 7.1 亏损 / 现金流为负的公司

- **早期创业公司**：营收微、亏损大 → DCF 显性期全负，结果高度依赖终值假设，几乎等于"猜"
- **替代方法**：
  - **VC 法**：预测退出时的营收 / 利润 × 目标退出倍数 / (1 + 目标 IRR)^N
  - **实物期权法**：把管理层未来选择权（放弃 / 扩张 / 转型）作为期权定价
  - **可比交易法**：找类似阶段公司融资估值
  - **PS / GMV 倍数**：互联网早期常用（但要小心讲故事）

### 7.2 周期性行业

- 单期利润高低不能代表稳态
- **替代**：用"周期中值利润 / 现金流"做 DCF，或用 EV/EBITDA 中周期均值

### 7.3 金融公司（银行 / 保险）

- 债务是原材料而非融资，WACC 概念不适用
- 用 **股权自由现金流（FCFE）** 或 **超额收益模型 / Residual Income Model**：

        权益价值 = 期初账面权益 + Σ (ROE - Ke) × 期初权益 / (1 + Ke)^t

### 7.4 房地产公司（重资产 / 高杠杆）

- FCFF 常年为负（拿地开发），估值周期长
- **替代**：NAV（净资产价值法）估值——项目现金流按项目周期贴现，累加得到 EV

### 7.5 极高增长 / 极不确定公司

- 显性期假设难以站得住
- **建议**：不主推 DCF，用相对估值 + 场景分析

## 8. DCF 与相对估值的关系

**相对估值 = 市场共识的 DCF 简化版**。DCF 与相对估值差异大时，先追问：

- 是市场对增长 / 风险预期不同？（可能是市场错）
- 还是我的 DCF 参数错？（可能是模型错）

**最佳实践**：DCF 得出内在价值区间 + 相对估值得出市场情绪区间，交叉验证。

## 9. 完整 DCF 工作流

1. **理解商业模式**：赚什么钱、多少客户、谁是护城河、竞争格局
2. **搭历史 3-5 年报表模型**：营收拆解、利润率、CapEx、WC
3. **假设显性期 5-10 年**：营收增长、利润率、CapEx / 营收、WC / 营收
4. **计算 FCFF 每年**
5. **估算 WACC**：CAPM 得 Ke，加权得 WACC
6. **终值双法**：永续增长 + 退出乘数
7. **贴现汇总得 EV**
8. **调整**：EV - 有息净债务 - 少数股东权益 + 多余现金 + 长期股权投资市值 = 股东权益价值
9. **除以股本**：得每股价值
10. **敏感性表**：WACC × g 二维矩阵，输出估值区间
11. **场景分析**：牛 / 中 / 熊三种情景各跑一遍
12. **与相对估值交叉验证**：EV/EBITDA、PE、PB、PS
13. **给出结论 + 假设透明化**：明确列出哪些假设是关键 driver

## 10. 简单示例

某成熟制造业公司：
- FCFF 显性期 5 年：¥10、11、12、13、14 亿
- WACC = 9%
- 永续增长 g = 3%
- 终值 = 14 × 1.03 / (0.09 - 0.03) = 240 亿
- 贴现和：10/1.09 + 11/1.09² + 12/1.09³ + 13/1.09⁴ + 14/1.09⁵ + 240/1.09⁵
- ≈ 9.17 + 9.26 + 9.27 + 9.21 + 9.10 + 155.94 = **202 亿**
- 减有息净债务 30 亿 → 股东权益 **≈ 172 亿**
- 若股本 10 亿股 → **每股 17.2 元**

敏感性（WACC × g）：结果区间 150-260 亿，视参数取值。
