# PptxGenJS Tutorial

## Overview

Reference guide for creating PPTX presentations from scratch using PptxGenJS.

## Workflow

1. Set up presentation (layout, author, title)
2. Add slides with varied content types
3. Apply styling and formatting
4. Export to .pptx file

## Output Contract

- Working PptxGenJS code producing a complete .pptx file
