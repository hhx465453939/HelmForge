# /finance-manager

调用 `finance-manager` skill，进入企业财务管理与财务分析模式。

## 何时使用

当你需要对一家公司（无论是自家还是标的公司）做以下任一分析时：

- **报表体检**：拿到一份财报，想知道"整体健康度如何、有没有坑"
- **财务比率**：偿债 / 盈利 / 营运 / 成长四大类比率的计算与解读
- **杜邦分解**：搞清楚 ROE 的高低究竟是靠净利率、周转还是杠杆
- **DCF 估值**：给一家（有正现金流的）公司做估值
- **现金流预警**：判断"这家公司现金流是不是要出问题了"
- **同行对比**：把公司比率放到行业基准里看真实位置
- **并购 / 授信 / 投前尽调**：需要一份结构化的财务健康诊断

## 输入建议

尽量一次提供以下要素，输出质量会显著更高：

1. 公司名称 / 行业 / 会计准则（中国企业会计准则 / IFRS / US GAAP）
2. 报表期间与审计状态（是否已审计）
3. 关键科目（最少：营收、净利润、总资产、总负债、经营性现金流；越全越好）
4. 分析目的（自查 / 尽调 / 授信 / 估值 / 并购）
5. （可选）3-5 家可比公司名单

信息不全也可以直接开始，`finance-manager` 会显式标注假设并给出置信度评估。

## 默认输出

- 财务健康一句话判断
- 关键财务比率表（含行业基准）
- 杜邦分解
- 风险点清单
- 短 / 中 / 长期改善建议
- 数据假设与复核提示（"⚠️ 本分析不替代审计"）
