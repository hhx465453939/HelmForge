# Work Product Templates

把输出做成律师、管理层、业务负责人能直接接手的文件。

## 1. Lawyer Briefing Memo

```markdown
## 委托方与目标
## 事实摘要
## 关键争点
## 我方最强路径
## 对方最强抗辩
## 证据现状与缺口
## 近期动作清单
## 需要律师确认的问题
```

## 2. Claim / Defense Matrix

```markdown
| 争点 | 我方主张 | 对方抗辩 | 我方反击 | 关键证据 | 风险等级 |
|---|---|---|---|---|---|
```

## 3. Evidence Checklist

```markdown
| 证据项 | 证明目的 | 当前是否掌握 | 可能来源 | 紧急度 |
|---|---|---|---|---|
```

## 4. Compliance Action Plan

```markdown
## 风险分级
## 立即停止事项
## 72小时动作
## 30天整改
## 对监管口径
## 对内口径
## 留痕与审计要求
```

## 5. Negotiation / Settlement Brief

```markdown
## 我方目标
## 最优结果
## 可接受结果
## 不可接受结果
## 可让步项
## 不可让步项
## 施压点
## 对方弱点
```

## 6. Fraud Recovery / Asset Tracing Memo

```markdown
## 资金路径与角色图
## 对方包装路径
## 明知/参与/受益证据
## 追偿抓手
## 补证建议
## 程序性下一步
```
