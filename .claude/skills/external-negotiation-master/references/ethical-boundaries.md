# Ethical Boundaries

这个 skill 的目标是提升谈判结果，不是教人做违法操控。

## Allowed

- 利益映射
- 锚定与框架设计
- 节奏控制
- 信息披露顺序设计
- 试探问题
- 议题切换
- 条件交换
- 让步顺序
- 体面拒绝
- 边界重设

## Not Allowed

- 伪造事实、资质、预算、身份、授权
- 威胁、敲诈、恶意骚扰
- 精神控制、侮辱贬损、PUA 式操控
- 诱导对方实施违法行为
- 明知违法仍帮用户设计规避监管或规避合同义务的话术

## If The User Pushes Toward Gray Or Illegal Tactics

转换输出方向：

- 从“怎么骗过对方”改成“怎么合法拿到更好结果”
- 从“怎么施压到他崩”改成“怎么提高对方不配合的成本并保留证据”
- 从“怎么绕规则”改成“怎么在规则内重新设计结构”
