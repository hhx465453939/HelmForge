# 三大报表分析

三大报表（资产负债表 / 利润表 / 现金流量表）是财务分析的基石。做任何比率、估值、预警之前，先确认三张表内部结构清晰、彼此勾稽、审计状态明确。

## 1. 资产负债表（Balance Sheet）

反映某一时点的资产、负债、所有者权益。核心恒等式：

    资产 = 负债 + 所有者权益

### 1.1 主要科目结构（中国企业会计准则）

**资产（按流动性排列）**

- 流动资产：货币资金、交易性金融资产、应收票据、应收账款、预付款项、其他应收款、存货、一年内到期的非流动资产
- 非流动资产：长期股权投资、投资性房地产、固定资产、在建工程、无形资产、开发支出、商誉、长期待摊费用、递延所得税资产

**负债**

- 流动负债：短期借款、应付票据、应付账款、预收款项（新准则改为合同负债）、应付职工薪酬、应交税费、一年内到期的非流动负债
- 非流动负债：长期借款、应付债券、长期应付款、递延所得税负债

**所有者权益**

- 实收资本（股本）、资本公积、盈余公积、未分配利润、库存股、其他综合收益、少数股东权益

### 1.2 常见科目质量判读

| 科目 | 需要警惕的信号 |
|---|---|
| 货币资金 | 账面很多，却仍在高息借款；受限资金占比高 |
| 应收账款 | 增速远高于营收增速；账龄结构恶化 |
| 存货 | 周转天数持续拉长；跌价准备计提不足 |
| 商誉 | 巨额商誉未减值、被收购方业绩承诺已未达标 |
| 在建工程 | 长期挂账不转固定资产（推迟折旧、粉饰利润）|
| 其他应收款 | 关联方大额往来款、无明确用途 |
| 短期借款 | 大幅上升 + 货币资金充裕（"存贷双高"经典红旗）|

## 2. 利润表（Income Statement）

反映一段期间的经营成果。核心结构（新准则）：

    营业收入
    - 营业成本
    - 税金及附加
    - 销售费用
    - 管理费用
    - 研发费用
    - 财务费用（收入为负）
    + 其他收益 / 投资收益 / 公允价值变动损益 / 信用减值损失 / 资产减值损失 / 资产处置收益
    = 营业利润
    + 营业外收入 - 营业外支出
    = 利润总额
    - 所得税费用
    = 净利润
    净利润归属：归属母公司股东 / 少数股东

### 2.1 利润质量判断

**"利润好"不等于"公司好"**，重点看：

- 营业利润占利润总额比例：健康公司应 > 80%（否则严重依赖政府补贴、投资收益、资产处置等非经常性损益）
- 毛利率的可持续性：单期跳升要追问原因（价格上涨？产品结构？会计估计变更？）
- 费用率：销售费用率 / 管理费用率 / 研发费用率的趋势
- 减值计提是否充分：应收账款、存货、商誉、长期资产减值

### 2.2 常见利润粉饰手法

1. **提前确认收入**：未发货就开票、发货未完成风险转移就确认；工程类"完工百分比"人为调高进度
2. **推迟确认费用**：应作费用化的支出资本化（在建工程 / 无形资产 / 开发支出）
3. **少提减值 / 折旧**：延长固定资产使用寿命、坏账准备计提比率降低、商誉减值久拖不计
4. **关联交易注水**：关联方以远高于市价采购公司产品
5. **一次性收益冲高**：处置资产 / 政府补贴 / 债务重组冲击利润
6. **调节存货成本**：先进先出与加权平均切换、"车间在制品"归集调节
7. **表外融资**：明股实债、售后回租、供应链金融保理不出表

判断信号：净利润高 但经营性现金流长期低于净利润，是最重要的红旗。

## 3. 现金流量表（Cash Flow Statement）

反映一段期间现金的流入流出，分三大活动：

- **经营活动现金流（CFO）**：主营业务产生的现金
- **投资活动现金流（CFI）**：购建 / 处置长期资产、对外投资
- **筹资活动现金流（CFF）**：借款、发债、增发、分红、还债

关系：

    期末现金余额 = 期初现金 + CFO + CFI + CFF + 汇率变动影响

### 3.1 三种现金流组合的公司画像

| CFO | CFI | CFF | 典型画像 |
|---|---|---|---|
| + | − | − | 成熟稳定型（自造血、还债、投扩产）|
| + | − | + | 成长扩张型（自造血 + 融资投建）|
| + | + | − | 衰退收缩型（收现金、卖资产、还债）|
| − | − | + | 初创烧钱型（亏损、投建、融资）|
| − | + | + | 危险信号（主业失血、卖资产借钱输血）|
| − | + | − | 濒危信号（失血 + 卖资产 + 还债，通常撑不久）|

### 3.2 关键推导：净利润 → 经营性现金流

    经营性现金流 ≈ 净利润 + 折旧摊销 + 减值 ± 营运资本变动

其中营运资本变动 = -(应收增加 + 存货增加 + 预付增加) + (应付增加 + 预收增加)

**如果 CFO 长期低于净利润，可能是**：
- 收入是应收账款堆出来的
- 存货堆积（可能滞销）
- 供应商话语权减弱，应付账款账期缩短
- 存在收入虚增

## 4. 三表勾稽关系（必须自查）

1. **利润表 → 资产负债表**：
   - 本期净利润（扣除分红）→ 未分配利润的净增加
   - 其他综合收益 → 其他综合收益（权益类）的净变动

2. **利润表 → 现金流量表**：
   - 净利润 + 折旧摊销 ± 营运资本变动 ≈ CFO
   - 差异过大要能解释（收入质量、减值、非现金损益等）

3. **资产负债表 → 现金流量表**：
   - 期末货币资金 - 期初货币资金 ≈ CFO + CFI + CFF + 汇率变动
   - 长期借款净变化 ≈ CFF 中的取得借款 - 偿还借款

**勾稽不上意味着报表错误 / 漏报 / 舞弊，任何后续分析都要暂停。**

## 5. 报表分析工作流

拿到报表后建议：

1. **明确审计状态**：已审计 / 未审计 / 内部管理报表？IFRS / CAS / US GAAP？
2. **勾稽自查**：三张表逻辑对齐一次
3. **纵向比较**：近 3-5 年同期科目变化 + 结构变化
4. **横向比较**：与同行 3-5 家关键比率对齐（见 `industry-benchmarks.md`）
5. **红旗清单**：把粉饰高发科目单独扫一遍（存贷双高、商誉、其他应收款、开发支出、在建工程）
6. **交叉验证**：与经营指标（产能利用率、客户数、订单积压）交叉印证

## 6. IFRS 与中国准则常见差异（会影响比率解读）

- 研发支出资本化的门槛：IFRS 相对严格，CAS 实操中"开发支出"资本化较宽
- 商誉减值 vs 摊销：都不摊销，只做减值测试；但减值时点判断可能不同
- 租赁：新租赁准则（IFRS 16 / CAS 21 修订）几乎全部资产化，负债率会显著上升
- 金融工具分类：SPPI 判断影响公允价值 vs 摊余成本

跨会计准则比较时，先对齐口径。

## 7. 常见坑与提醒

- 分部报表（segment reporting）里主业与副业混在一起时，需要单独看
- 少数股东权益容易被忽略，导致 ROE 分子分母口径错配
- 一次性 / 非经常性损益要剔除后再看趋势
- 汇率折算影响：合并报表汇率变动可能扭曲增长率
- 会计政策变更 / 追溯调整：跨期比较时要看是否重述
