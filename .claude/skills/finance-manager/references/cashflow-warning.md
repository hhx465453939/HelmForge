# 现金流预警

利润可以粉饰，现金流很难。现金流预警的核心思路：把"账面看起来还行"和"资金链要断"之间的时间差找出来，越早识别越有回旋余地。

本文件列出 8 类核心预警信号、阈值和触发动作，供财务健康诊断快速扫描使用。

## 预警信号 1：经营性现金流与净利润长期背离

**指标**：CFO / 净利润

**阈值**：
- 健康：≥ 0.9（长期均值）
- 关注：0.5 - 0.9（连续 1 期）
- 警报：< 0.5（连续 2 期以上），或 CFO 为负而净利润为正

**可能原因**：
- 收入是应收账款堆出来的
- 存货堆积（可能滞销、跌价准备不足）
- 供应商话语权减弱，应付账款账期缩短
- 存在收入虚增 / 提前确认

**触发动作**：
- 追查应收账款账龄、大客户名单、销售政策变化
- 存货结构与库龄分析
- 与同行 CFO / 净利润比率对比
- 若持续 2 年以上，把这个视为红色预警，不用 DCF 做估值时先修正 FCF 假设

## 预警信号 2：现金短债比过低

**指标**：货币资金 / 短期有息负债

**阈值**：
- 健康：≥ 1.0
- 关注：0.5 - 1.0
- 警报：< 0.5

**含义**：现金能否覆盖 1 年内到期的有息负债。这是"三道红线"里的核心指标之一，非房企同样适用。

**触发动作**：
- 分析 12 个月内到期债务明细
- 评估授信额度、发债窗口、股东借款可能性
- 若同时 CFO 也在恶化 → 高度警惕流动性危机
- 检查是否存在受限资金（保证金 / 冻结）虚增账面货币资金

## 预警信号 3：应收账款异常增长

**指标**：应收账款增速 / 营收增速

**阈值**：
- 健康：< 1.2
- 关注：1.2 - 1.5
- 警报：> 1.5

**含义**：应收账款涨得比收入快很多，意味着"卖出去的东西越来越难收回来"。

**结合看**：
- 应收账款周转天数是否连续拉长？
- 单一大客户占比是否上升？
- 账龄结构是否恶化（1 年以上占比上升）？
- 坏账准备计提是否与账龄结构匹配？

**触发动作**：
- 索取前 5 / 10 大客户明细、应收账龄表
- 与销售政策变化对齐（是否为冲收入放宽账期）
- 现金流量表里"销售商品收到的现金 / 营业收入"是否在 1.1-1.2 附近（正常水平）
- 若比率 < 1.0 且连续下滑，警惕收入注水

## 预警信号 4：存货堆积

**指标**：存货增速 / 营收增速；存货周转天数变化

**阈值**：
- 健康：存货增速 ≤ 营收增速
- 关注：存货增速 > 营收增速 20%
- 警报：存货增速 > 营收增速 50%，或存货周转天数拉长 30% 以上

**可能原因**：
- 销售不达预期
- 库存管理能力下降
- 原材料囤积（可能对赌价格）
- 存货虚增（舞弊）

**触发动作**：
- 查看存货明细（原材料 / 在产品 / 产成品）结构变化
- 跌价准备计提比例是否与库龄匹配
- 期后销售情况印证（若跨期销售数据可得）
- 如属于消费品 / 时尚品，警惕过季存货

## 预警信号 5：投资活动现金流异常放大

**指标**：CFI（投资活动现金流出）/ 营收

**阈值**：
- 常规扩产：0 - 15%（视行业）
- 关注：15 - 30%
- 警报：> 30%（非典型大规模 CapEx）

**风险场景**：
- 大规模并购（尤其是溢价并购形成高商誉）
- 大规模在建工程（可能推迟折旧、粉饰利润）
- 关联方资产收购（利益输送）
- 投资性理财 / 股权投资异常放大

**触发动作**：
- 查看并购标的的业绩承诺完成情况
- 商誉减值测试是否合理
- 在建工程转固节奏是否与实际投产匹配
- "购建固定资产 / 无形资产 / 其他长期资产支付的现金" 与新增固定资产、无形资产账面数是否勾稽

## 预警信号 6：筹资依赖度上升

**指标**：CFF 净流入 / CFO；有息负债增速 / 营收增速

**阈值**：
- 健康：CFO 覆盖 CapEx 有余，CFF 为负（还债 / 分红）
- 关注：CFF 净流入 > 0，且有息负债增速 > 营收增速 20%
- 警报：CFF 净流入 > CFO，且连续 2 期以上；有息负债快速堆积

**含义**：主业不能自给自足，靠外部输血维持。

**触发动作**：
- 判断融资是否用于扩张（合理）还是覆盖亏损（危险）
- 评估融资渠道的可持续性（银行授信剩余、市场发债能力、大股东支持）
- 计算"利息 / CFO"，看主业能否覆盖利息

## 预警信号 7：利息覆盖恶化

**指标**：利息保障倍数 = EBIT / 利息费用

**阈值**：
- 健康：> 3
- 关注：1.5 - 3
- 警报：< 1.5
- 危险：< 1（利润不够付利息）

**触发动作**：
- 如接近 1.5，评估未来 12 个月是否有利率调整、债务到期、增量借款需要
- 若已经 < 1，需要立即评估"止血"方案：处置资产 / 债务重组 / 增资
- 检查是否存在明股实债、优先股股息等表外融资成本

## 预警信号 8：应付账款异常延长（供应商话语权下降）

**指标**：应付账款周转天数

**阈值**：
- 健康：稳定或缓步上升
- 关注：单期上升 20% 以上
- 警报：上升 30% 以上或已远超行业平均

**注意**：应付延长在早期可能是"占用供应商资金"的好事（如强势零售商），但过长 → 供应商开始要求现款 or 停止供货 → 供应链危机。

**触发动作**：
- 结合货币资金和短期借款一起看：若同时货币资金减少 + 应付延长 → 明显现金紧张
- 询问是否已有供应商停止供货、要求预付款
- 供应商集中度分析（若集中度高，风险更大）

| 信号 | 权重 | 你的公司 |

| CFO / 净利润 | 25% | |
| 现金短债比 | 20% | |
| 应收账款增长 vs 营收增长 | 10% | |
| 存货增长 vs 营收增长 | 10% | |

3 个以上红灯 → 高危；2 个红灯 + 3 个黄灯 → 中危；仅零星黄灯 → 轻度关注。

## 现金流预警的分级响应

### L1 关注（1-2 个黄灯）

- 持续监控，季度追踪
- 与同行对比进一步确认是否行业性问题

### L2 警报（2-3 个红灯或 3+ 个黄灯）

- 立即启动 12 个月现金流预测（周度或月度）
- 与主要银行沟通授信状态
- 加速应收回收 / 减少 CapEx / 库存去化
- 与股东沟通可能的增资 / 借款

### L3 危机（4+ 个红灯或核心红灯连续 2 期）

- 启动应急现金管理：冻结非必要支出、盘活闲置资产
- 与债权人沟通展期 / 重组
- 评估资产处置 / 战略投资者引入
- 法务评估破产重整可能性和边界（联动 global-legal-counsel）
- 与关键客户 / 供应商沟通稳定预期

## 现金流预测：12 个月滚动预测模板

无论处于什么等级，只要出现预警，都建议做一份 12 个月滚动现金流预测：

| 月份 | 期初现金 | 经营流入 | 经营流出 | 投资净流出 | 融资净流入 | 期末现金 | 现金短债比 |
|---|---|---|---|---|---|---|---|
| M1 | | | | | | | |
| M2 | | | | | | | |
| ... | | | | | | | |
| M12 | | | | | | | |

- 关键假设：营收增长、账期变化、CapEx、还款计划、股权融资时点
- 敏感性：营收 -10% / -20% / -30% 三种情景
- 触发线：任一月份期末现金 < 3 个月经营支出 → 立即行动
